# Leecode 刷题项目

这是一个用于 **LeetCode（力扣）算法题练习** 的个人项目仓库，主要用于记录刷题过程、解题思路以及代码实现，方便日后复习与总结。

---

## 📌 项目目的

* 系统性刷 LeetCode 算法题
* 夯实 **数据结构与算法基础**
* 沉淀高质量、可复用的解题模板
* 为面试、竞赛和工程实践打好基础

---

## 🧠 题目范围

当前计划覆盖（持续更新）：

* 数组 / 字符串
* 双指针
* 哈希表
* 栈 / 队列
* 链表
* 二叉树 / N 叉树
* DFS / BFS
* 回溯算法
* 动态规划（DP）
* 贪心算法
* 二分查找
* 图论基础

---

## 📂 项目结构示例

```text
leecode/
├── array/            # 数组相关题目
├── string/           # 字符串相关题目
├── linked_list/      # 链表
├── tree/             # 树
├── dp/               # 动态规划
├── graph/            # 图论
├── backtracking/     # 回溯
├── binary_search/    # 二分查找
└── README.md
```

> 每道题通常包含：
>
> * 题目链接
> * 解题思路说明
> * 时间 / 空间复杂度分析
> * 代码实现

---

## 📝 解题规范（建议）

* 清晰写出 **核心思路**，避免只贴代码
* 标注 **时间复杂度 & 空间复杂度**
* 复杂题目给出 **优化过程**（如从暴力到最优解）
* 重要题目标注 ⭐，便于后期复习
